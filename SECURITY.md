# Security Policy

## Supported Versions
| Version | Supported          |
|---------|--------------------|
| 5.0.x   | ✅ Fully supported |
| 4.0.x   | ✅ Security fixes  |
| < 4.0   | ❌ No support      |

## Reporting Vulnerabilities
Email: security@hyperboost.dev
Response time: 48 hours

## Safety Warnings
- Level 3-5 require root access
- Thermal unlocker can damage hardware
- Always backup before using
- Use at your own risk

## Best Practices
1. Backup before optimization
2. Test one module at a time
3. Read documentation first
4. Don't use thermal unlocker unless you understand the risks
